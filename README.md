# bookmarkletter

bookmarkletter convert code to **bookmarklets**

## Installation

```
npm install bookmarkletter -g
```

## Usage

```sh
$ bookmarkletter code.js
# or
$ cat code.js | bookmarkletter
```

## API

bookmarkletter allow to use as node module.

```
npm install bookmarkletter
```
